# Survey Data Analysis Project

This project demonstrates an end-to-end survey data analysis workflow using anonymized academic and institutional survey data.

## Survey Type
- Student feedback surveys
- Program evaluation surveys
- User experience surveys

## Question Formats
- Likert-scale responses
- Multiple-choice questions

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Microsoft Excel
- Power BI

## Key Tasks
- Data cleaning and validation
- Descriptive and exploratory analysis
- Visualization of survey responses
- Insight-driven reporting for non-technical stakeholders

## Files
- sample_survey_data.csv – anonymized survey dataset
- Survey_Data_Analysis_Project_Mohammed_Saadullah.docx – project report
- Survey_Analysis_Notebook_Guide.docx – analysis workflow and sample code

> Note: All data is anonymized and used for demonstration purposes only.
